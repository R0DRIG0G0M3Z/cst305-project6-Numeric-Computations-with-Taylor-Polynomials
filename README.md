# CST 305 Benchmark 6 Numeric Computations with Taylor Polynomials
## Rodrigo Gomez

## Description

This project solves different differential equations using Taylor polynomials and simple numerical methods.

There is three main parts in the project:

Part 1: Taylor polynomial approximations of differential equations  
Part 2: Numerical solution of a second-order equation  
Part 3: A simple model of computer system performance  

---

## Requirements

Python and packages numpy, matplotlib  

You install them using:
pip install numpy matplotlib

---

## How to Run

1. Open the folder in VS Code
2. Open the file main.py
3. Run the program using python main.py
